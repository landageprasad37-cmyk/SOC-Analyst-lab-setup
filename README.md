# Enterprise SOC Analyst Home Lab

## Overview

This repository documents the development of my enterprise-style Security Operations Center (SOC) Analyst home lab. The goal is to build practical cybersecurity skills by creating and managing a realistic virtual enterprise environment using industry-standard technologies.

The lab focuses on hands-on learning through system administration, network security, log analysis, threat detection, and security monitoring. Every configuration, challenge, and solution is documented to demonstrate technical knowledge, troubleshooting skills, and continuous progress.

---

## Learning Objectives

- Build and manage a virtual enterprise environment
- Understand networking and firewall administration
- Configure secure network infrastructure
- Develop Linux and Windows administration skills
- Learn enterprise authentication and directory services
- Collect, analyze, and investigate security logs
- Practice threat detection and incident analysis
- Document configurations and troubleshooting procedures
- Develop a professional cybersecurity portfolio

---

## Learning Roadmap

### Month 1 — Infrastructure & Networking

Build the foundation of the enterprise lab by configuring virtualization, networking, firewalls, and core operating systems.

### Month 2 — Enterprise Administration

Deploy enterprise services, configure identity management, strengthen endpoint security, and practice system administration.

### Month 3 — Security Operations

Implement centralized logging, security monitoring, detection engineering, threat investigation, and incident response workflows.
